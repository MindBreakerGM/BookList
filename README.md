Online library, adding books, writing reviews, etc.

Technologies
- React.js
- Node.js
- Express
- JWT
- MongoDB

### Start project local

### Frontend
```
npm install
```

```
npm start
```

### Backend

```
npm install
```
Create .env file

Enter db_url,port,jwt_private_token,mailPassword,mailLogin,testJwt,loginUser,passwordUser

```
node app.js
```
### Start basic unit tests for backend
```
npm test
```
