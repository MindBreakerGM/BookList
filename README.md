### BookList
Online library, adding books, writing reviews, etc.

Technologies
- React.js
- Node.js
- Express
- JWT
- MongoDB

### Start project local

### Frontend
```
npm install
```

```
npm start
```

### Backend

```
npm install
```
Create .env file

Enter DATABASE_URL,secret,PORT

```
node app.js
```
### Start basic unit tests for backend
```
npm test
```
